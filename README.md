# cofflix (2020)
취향 선택을 통한 카페 추천 사이트

## Service introduction
내가 원하는 취향의 카페 분위기, 메뉴, 야외, 주차여부 등 선택을 통해 추천을 받는 시스템

## Main Function
1. 지도 기능을 결과에 보여주어 추천된 카페의 위치 연결
2. 취향 선택을 통해 고객에 맞춘 결과를 도출하는 기능을 구현
